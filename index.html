O sistema possui gráficos

  • Gráfico mostrando produtos que precisam ser comprado
Características que os clientes mais gostam:
  • Alta velocidade do sistema
  • O sistema permite realizar vendas rápidas, evitando filas de espera
  • Relatório de vendas, podendo saber quais produtos vendem mais
